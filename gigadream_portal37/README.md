pip install -e .

## Usage

```bash
# Full animated ritual (default)
gigadream

# With sound layer (beach + orbital + starjet + portal vibes)
gigadream --sound

# Heavy glitch storm
gigadream --mode glitch --intensity 0.55 --sound

# Clean static transmission
gigadream --mode static

# Secret PORTAL 37 overload mode (hidden)
gigadream --mode secret --sound

# Export the final frame
gigadream --export
```

**Flags**
- `--sound` — Engage the ambient sound layer
- `--intensity` — Glitch strength (0.0–1.0)
- `--duration` — How long the transmission runs
- `--color full|minimal|none`
